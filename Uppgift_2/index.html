Vad fyller man en tabbell med?

Att strukturera sitt pressentationsdata kan vara knepikgt med HTML och ett vanligt
misstag kan vara att att välja ett html-element för att få saker ligga visuellt rätt.
Tabell elementet <table> kan vara ett bra exempel på när html-element används symantiskt fel.
Iställlet för att pressentera tabbell-data så använsds tabellen till för att göra en layout-pressentation.
Detta var väldigt vanligt i htmls barndom innan css gjortde sitt intåg.
